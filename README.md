# 🫕 MyRaclette

Application de gestion de collection pour tes soirées raclette.

## Lancer en local

```bash
npm install
npm run dev
```

Ouvre http://localhost:5173

## Déployer sur Vercel

1. Pousse ce dossier sur GitHub
2. Connecte le repo sur https://vercel.com
3. Vercel détecte Vite automatiquement → clique sur **Deploy**

C'est tout ! 🎉
